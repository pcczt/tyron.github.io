深入分析Java Web技术内幕（修订版）
许令波 / 2014-8-1 电子工业出版社 / 7.5 分
2018-11-14 / java 计算机 编程 / ★★★☆☆ 还行
刚拿到书，感觉写的很不错，越看到后面越质疑我之前的观点。此书写的面的确很全，可谓面面俱到，但是面面俱到的反面就是都不深入，与之标题相差甚远，《浅析Java Web技术内幕》更合适。 纵观全书，本书更适合初级Java开发人员对JavaWeb做进一步的了解，作者对单个模块的理解上挺深入的，但是想要对各个模块有深一步的理解的话还需要再看其他的书了。
